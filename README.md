This package contains code for Hamiltonian and Liouvillian learning based on ``Olsacher et al 2025 Quantum Sci. Technol. 10 015065``

Example notebooks can be found in the ``../examples`` folder.

The ``requirements.txt`` was created via ``pip freeze > requirements.txt``.
